Overview
--------

FG-Vector Sampler is a geometry-first molecular cluster pre-sampling framework.
It is designed for rapid, energy-free generation of diverse cluster candidates
using center-of-mass anchored functional-group vectors.

Key features
------------

* Multi-monomer sampling with arbitrary stoichiometry
* Beam-search candidate generation
* Logging and provenance support for research workflows
* Export of XYZ and JSON metadata
* Parallel execution for larger sampling campaigns
